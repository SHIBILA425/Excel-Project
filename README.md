# E-Commerce Performance Dashboard - Excel
## Project Overview
This project presents an E-Commerce Performance Dashboard built in Microsoft Excel to analyze sales performance, customer behavior, and profitability across different markets. The dashboard provides insights into revenue trends, sales channels, payment methods, and regional performance to support data-driven business decisions.
## Tools & Technologies
- **Microsoft Excel**
- **Power Query** – Data cleaning and transformation
- **Pivot Tables & Pivot Charts** – Data analysis
- **Excel Dashboard** – Interactive visualization
## Dataset
The dataset contains e-commerce transaction data including:
- Orders
- Revenue
- Profit
- Items sold
- Payment methods
- Countries
- Sales channels
- Return status
  
This data helps evaluate business performance across regions, platforms, and customer purchasing behavior.
## Data Preparation
The following steps were performed to prepare the dataset:
- Cleaned and transformed raw data using **Power Query**
- Handled missing values and inconsistencies
- Created calculated columns for analysis
- Structured the dataset into an **Excel table** for dynamic dashboard updates
## Key KPIs
The dashboard tracks the following metrics:
- **Total Revenue**
- **Total Profit**
- **Total Orders**
- **Items Sold**
- **Return Rate**
## Key Insights
- Sales peak during **October–November**, indicating strong seasonal demand.
- **India generates the highest revenue and profit**, followed by the USA.
- The **Web channel drives the majority of sales**, while the mobile app has lower engagement.
- **Card payments are the most widely used payment method**.
- Return rates are relatively low, indicating good customer satisfaction.
## Business Recommendations
- Focus marketing efforts on **high-performing markets like India and the USA**.
- Improve the **mobile app experience** to increase mobile purchases.
- Promote **digital payment methods** such as UPI and wallets.
- Analyze returned orders to identify and fix potential product or delivery issues.
- Plan **inventory and marketing campaigns around peak sales months**.
## Project Outcome
This project demonstrates how Excel dashboards can transform raw e-commerce data into actionable insights. 
The analysis helps businesses understand sales patterns, improve customer experience, and optimize revenue strategies.
